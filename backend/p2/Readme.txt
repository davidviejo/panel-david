El Inventario Funcional: Qué hace exactamente cada herramienta (Tu "Diccionario").
El Framework de Trabajo (Workflow): En qué orden usarlas para ejecutar una consultoría SEO profesional de principio a fin.
PARTE 1: DICCIONARIO DE HERRAMIENTAS (¿Para qué sirve cada una?)
Aquí tienes tus 52 módulos explicados según las pestañas de tu menú lateral:
1. ESTRATEGIA (Crecimiento)
📈 EcoTrends (Google Trends): [NUEVO] Radar de mercado en tiempo real. Monitoriza qué está "caliente" ahora mismo en Economía, Salud o Tech para hacer Newsjacking (escribir noticias de última hora).
🚀 SEO Suite / Cluster: Tu "cerebro". Le das 1.000 keywords desordenadas y te las agrupa por intención de búsqueda. Úsala al principio de cualquier proyecto. La segunda pestaña de esta herramienta saca todas las SERP analizando palabras, imágenes y encabezados dando un posible resultado para encontrar la estructura de contenido perfecta.
💡 KW Discovery: Tu "tormenta de ideas". Le das una palabra ("café") y te devuelve 300 sugerencias del autocompletar de Google.
🧠 KW Intent: Clasifica keywords en "Comprar" (Transaccional) vs "Aprender" (Informacional) para saber si crear una ficha de producto o un post.
⚖️ Content Gap: Espía a la competencia. Te dice qué palabras usan ellos que tú no.
⚖️ SERP Overlap: El juez de canibalización. Te dice si debes crear dos artículos distintos para dos keywords o uno solo.
📊 SERP Analyzer: Benchmarking. Te dice cuántas palabras, imágenes y encabezados tiene el Top 10 para que sepas cuánto escribir.
2. AUDITORÍA (Salud del Sitio)
🕸️ Auditor Técnico: Escáner profundo. Revisa H1, Meta Descriptions y Thin Content de un Sitemap.
✅ Checklist Auto: Genera un Excel con columnas "SI/NO" (Schema, Geo, etc.) para rellenar informes administrativos rápido.
⚡ Fast Status: Comprueba si 1.000 URLs dan error 404 o 200 OK en segundos (sin descargar el HTML).
🔍 Index Guard: Verifica si tienes noindex o canonicals mal puestos que impiden indexar.
🔍 Index Checker: Comprueba si tus URLs aparecen en los resultados de búsqueda (Bing/DDG/Google).
🤖 Robots.txt: Verifica si el archivo robots.txt está bloqueando a Googlebot en URLs específicas.
🛡️ E-E-A-T Trust: Evalúa la "confiabilidad" (autores, políticas legales, citas externas).
🕵️ Tech Spy: Te dice qué CMS (WordPress, Shopify) y Plugins usa la competencia.
📡 SSL & Headers: Revisa la seguridad, caducidad del SSL y caché del servidor.
3. ON-PAGE (Contenidos)
🧠 NLP Analyzer: Inteligencia Artificial. Te dice el sentimiento, subjetividad y entidades (personas/marcas) de un texto.
🎯 KW Prominence: Te dice si la palabra clave está en "zonas calientes" (Inicio del título, H1, primer párrafo).
🎯 Snippet Target: Te ayuda a redactar el párrafo perfecto para robar la "Posición 0" de Google.
📋 KW Checker: Valida si el redactor ha incluido la lista de palabras obligatorias que le pediste.
👶 Readability: Te marca en colores las frases demasiado largas o difíciles de leer.
📝 Draft Editor: Un editor en blanco para pegar borradores y ver métricas SEO en tiempo real.
👯 Duplicity: Compara dos URLs para ver si hay plagio o contenido duplicado interno.
📉 Content Decay: Te dice qué artículos antiguos han "caducado" y necesitan actualización.
4. ARQUITECTURA (Estructura)
🕸️ Link Graph: Visualización gráfica de bolas y líneas de tu enlazado interno.
📉 Click Depth: Te dice a cuántos clics de la Home está cada página (Nivel 1, 2, 3...).
🧮 LinkRank: Calcula matemáticamente qué página tiene más autoridad interna (PageRank).
🗿 Link Sculpting: Cuenta enlaces salientes para evitar la dilución de autoridad.
⚓ Anchor Audit: Analiza los textos de los enlaces para evitar sobre-optimización ("keyword stuffing" en enlaces).
🏝️ Orphan Hunter: Encuentra páginas que existen en el Sitemap pero no tienen enlaces internos.
🗺️ Header Map: Muestra el esqueleto (H1-H3) de tu web vs la competencia en columnas.
🔗 Link Opps: Encuentra oportunidades automáticas para poner enlaces internos en textos ya escritos.
🚑 Link Health: Encuentra enlaces rotos (404) dentro de una página.
🔀 Redirect Path: Rastrea cadenas de redirecciones (A->B->C) para limpiarlas.
5. WPO (Velocidad)
⚡ WPO Lab: Mide el tiempo de respuesta del servidor (TTFB) y peso total.
🖼️ Image Auditor: Encuentra imágenes pesadas (>100kb) o sin etiquetas ALT.
🔢 Code Ratio: Te dice si tu web es todo código y poco texto (Code Bloat).
6. UTILIDADES Y NEGOCIO
💰 ROI Projector: Calculadora para vender proyectos (proyección de ingresos).
📝 Meta Gen: Crea Títulos y Descripciones masivamente usando plantillas.
🔥 CTR Magnet: Analiza si tus títulos son "clickbait" o aburridos.
📏 Pixel Counter: Verifica si Google cortará tu título con puntos suspensivos (...).
🎨 SERP Sim: Previsualizador visual de cómo queda en Google.
🏗️ Schema Gen: Crea el código JSON para FAQs automáticamente.
🧬 Schema Extract: Saca los datos (precio, stock) del código de la competencia.
🌍 Hreflang: Valida configuraciones multi-idioma.
🕷️ Bot Sim: Detecta si el servidor bloquea a Google o si haces Cloaking.
🏷️ UTM Builder: Crea URLs trackeables para campañas.
📍 Local Mixer: Genera combinaciones de palabras clave para SEO Local.
📍 NAP Validator: Comprueba consistencia de Nombre/Tlf/Dirección.
🕵️ Outreach: Encuentra emails de contacto en webs para pedir enlaces.
🗑️ Expired: Encuentra dominios caducados en enlaces salientes.
🆚 Diff Checker: Compara el código de dos versiones de una web (Staging vs Live).
🗺️ Sitemap Gen: Crea un XML rastreando una web que no lo tenga.
📄 AI Prep: Extrae texto limpio para dárselo a ChatGPT.
📢 Social: Previsualiza cómo queda el link en Twitter/LinkedIn.
🕵️ Dorks: Genera enlaces de búsqueda para auditoría manual rápida.
🗂️ Log Analyzer: Analiza archivos de logs del servidor.
PARTE 2: EL FRAMEWORK DE TRABAJO (Tu Metodología)
Como consultor, no usas todo a la vez. Usas flujos de trabajo. Aquí tienes 5 Playbooks usando tu suite:
PLAYBOOK A: La Auditoría Inicial (Día 1 con el cliente)
Objetivo: Encontrar los errores graves rápido.
🕵️ Dorks: Busca páginas hackeadas o HTTP olvidadas.
🕸️ Auditor Técnico: Escanea el Sitemap para ver el estado general (H1, Metas).
⚡ Fast Status: Si el cliente te da una lista de URLs viejas, comprueba cuáles están vivas.
🕷️ Bot Sim: Comprueba que no estén bloqueando a Google.
📉 Click Depth: ¿Está todo el contenido importante a menos de 3 clics?
💰 ROI Projector: Prepara la propuesta económica con la proyección de crecimiento.
PLAYBOOK B: Estrategia de Contenidos (News & Evergreen)
Objetivo: Planificar el calendario editorial mezclando contenido estable y noticias virales.
📈 EcoTrends: Escanea el mercado (filtro "Economía" o tu sector). ¿Hay alguna noticia rompiendo el mercado hoy? Si SÍ -> Escribe rápido (Newsjacking).
💡 KW Discovery: Si no hay noticias, busca temas Evergreen ("mejores hipotecas").
🚀 SEO Suite (Cluster): Agrupa esas ideas.
🧠 KW Intent: Separa lo informativo de lo comercial.
⚖️ Content Gap: Mira qué le falta a tu competencia.
📄 AI Prep: Pásale los datos a la IA para redactar
PLAYBOOK C: Optimización On-Page (Antes de Publicar)
Objetivo: Publicar contenido perfecto.
📝 Draft Editor: Pega el texto del redactor.
📋 KW Checker: Verifica que ha incluido las palabras del Content Gap.
👶 Readability: Asegúrate de que no sea un ladrillo difícil de leer.
🔥 CTR Magnet: Pule el Título para que la gente haga clic.
📏 Pixel Counter: Asegúrate de que no se corte en Google.
🔗 Link Opps: Busca en tu blog antiguo dónde poner enlaces hacia este nuevo post.
PLAYBOOK D: Arquitectura y Autoridad (Mantenimiento Mensual)
Objetivo: Mejorar el flujo de fuerza.
🧮 LinkRank: Descubre qué páginas acumulan demasiada fuerza y cuáles tienen poca.
🕸️ Link Graph: Visualiza si hay islas desconectadas.
🏝️ Orphan Hunter: Encuentra páginas que se te han olvidado enlazar.
🗿 Link Sculpting: Revisa si estás perdiendo fuerza por tener demasiados enlaces en el footer.
🚑 Link Health: Arregla enlaces rotos dentro de tus artículos top.
PLAYBOOK E: Limpieza y Migraciones (Técnico Hardcore)
Objetivo: Arreglar una web rota o migrarla.
📉 Content Decay: Identifica contenido viejo para borrar o actualizar.
🆚 Diff Checker: Compara la web de desarrollo con la real para asegurar que no faltan etiquetas.
🔀 Redirect Path: Asegura que las redirecciones sean directas (A->B) y no cadenas (A->B->C).
🛠️ Migration Helper: Genera el código .htaccess masivo automáticamente.
🔍 Index Checker: Verifica que las nuevas URLs se están indexando.
